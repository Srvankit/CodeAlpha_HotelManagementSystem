# 🏨 Hotel Reservation System

A Java-based Hotel Reservation System developed as part of the **CodeAlpha Java Programming Internship**.

This project simulates a hotel booking environment where users can search rooms, make reservations, manage bookings, cancel reservations, and view payment summaries through a menu-driven console application.

---

## 🚀 Features

- 🔍 Search available rooms by category
- 🏨 View all available rooms
- 📋 Make hotel reservations
- ❌ Cancel reservations
- 💳 View payment summary
- 📂 Booking history logging
- 📊 Session statistics tracking
- 💾 File handling for data persistence

---

## 🛠️ Tech Stack

- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework
- ArrayList
- File Handling
- Exception Handling

## 📸 Functionalities

### 1️⃣ View Rooms

Displays all currently available rooms.

### 2️⃣ Search Rooms

Search rooms by category.

Examples:

- Standard
- Deluxe
- Suite

### 3️⃣ Book Room

Allows users to reserve an available room.

### 4️⃣ View Reservations

Displays all active reservations.

### 5️⃣ Cancel Reservation

Cancels an existing reservation.

### 6️⃣ Payment Summary

Displays the total amount for all active reservations.

### 7️⃣ Session Summary

Displays:

- Total bookings made
- Total cancellations
- Current active reservations
